# 👍 CSS(Cascading Style Sheets)

## ⚡ CSS Basic Syntax

```
선택자 {
    속성: 값;
    속성: 값;
}
```

### 선택자(Selector)

: 선택자는 스타일을 적용할 대상입니다. 태그, ID, Class 등을 선택할 수 있습니다.

```
/* 태그를 선택자로 했을 때 */
p {
    color: red;
}

/* 태그를 ID로 했을 때 */
#title {
    color: blue;
}

/* 태그를 Class로 했을 때 */
.item {
    color: green;
}
```
